How to run the program:
user the "make" command to make the program
use ./ASN-1 to run the custom bash terminal

while using the custom bash terminal you should be able to use all
commands as was previously available such as vim, cat, ls ...

There are some additional commands:

"history"
This command will display the ID PID, and Command of your ten most recent commands.

"!!"
will execute the most recent command run by the user.

"!N" -> Where N is a number in the realm of 1-10
This command will the Nth command run where 1 is the most recent and 10 is the oldest.
